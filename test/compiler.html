This is a test page for my Regular Expression Engine (GitHub). You can use it to convert a regular expression to a transition table, to be revealed underneath the input box.

The rows correspond to the states of a DFA (Wikipedia). The columns that are labeled with character ranges, correspond to outgoing edges. The row with a teal id-cell, is the ‘current state’. This is the start state of the DFA.

The dimmed cells refer to state 0, which corresponds to the ‚ä•-expression. It does not match any string at all. The state with id 1 corresponds to the ‚ä§-expression. It matches all strings. The state with id 2 corresponds to the Œµ-expression, It matches only the empty string, and state-3 matches any string of length 1.
